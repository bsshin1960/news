# Plan & Help - 뉴스 브리핑 & TTS 재생 PWA 앱

본 문서는 아침 뉴스 요약 브리핑 및 한국어 TTS(음성 합성) 재생 기능을 갖춘 모던 PWA 애플리케이션의 최종 제작 과정과 도움말을 한글로 기술한 문서입니다.

---

## 1. 프로젝트 제작 과정 (Development Record)

### [1] 기술 스택 선정 및 개발 환경 구축
- **핵심 언어**: HTML5, CSS3, ES6+ JavaScript
  - 네이티브 앱 수준의 설치성 및 브라우저 기본 Web Speech API와의 원활한 통합을 위해 최신 웹 표준 기술을 사용하여 개발했습니다.
- **빌드 도구**: Vite (Vanilla JS 개발 환경)
  - 번들 크기를 최적화하고 빠른 로컬 개발 서버 및 정적 리소스 서빙을 위해 적용했습니다.

### [2] UI/UX 및 디자인 시스템 구현 (테마 스위치 탑재)
- **기본 라이트 모드 (Default Light Theme) 채택**:
  - 밝고 투명한 화이트/소프트블루 그라데이션 레이아웃을 기본값으로 채택하여 눈의 피로를 덜고 맑은 아침 브리핑 느낌을 강조했습니다.
- **다크/라이트 테마 전환 스위치**:
  - 상단 헤더 영역에 **테마 스위치 버튼(달/해 아이콘)**을 배치하여, 클릭 한 번으로 미세한 색상 트랜지션 애니메이션과 함께 다크모드 <-> 라이트모드 변환이 매끄럽게 수행되도록 설계했습니다. 테마 설정은 로컬저장소에 보존됩니다.
- **퀵 설정(⚙️) 버튼 탑재 (최신 업데이트)**:
  - 우측 상단 테마 변경 버튼 바로 왼쪽에 동일한 스타일의 **설정(⚙️) 단축 아이콘**을 신설하여, 메뉴바를 탐색하는 번거로움 없이 클릭 한 번으로 즉시 뉴스 상세 설정 모달을 띄울 수 있도록 편의성을 극대화했습니다.
- **글래스모피즘(Glassmorphism) 테마**:
  - `backdrop-filter: blur(12px)`를 적용하여 투명하면서도 세련된 프리미엄 윈도우 스타일의 반투명 카드를 연출했습니다.
- **풀다운(Dropdown) 메뉴바**:
  - 화면 상단에 키보드 조작성을 감안한 파일(F), 편집(E), 도움말(H) 풀다운 메뉴를 배치하여 데스크톱 GUI 환경의 직관성을 살렸습니다.
- **마이크로 애니메이션**:
  - 뉴스 카드 마우스 오버 시의 부드러운 승강(Lift) 효과, 낭독 중인 카드 테두리의 무지개색 네온 그라데이션 라이팅 효과, 하단 재생바의 CSS 웨이브폼 오디오 비주얼라이저 효과 등을 추가하여 인터랙션 품질을 극대화했습니다.

### [3] 뉴스 수집 및 AI 요약 비즈니스 로직
- **언론사 보도 신뢰성 확보**:
  - Gemini AI가 요약을 생성할 때 **"연합뉴스, YTN, KBS, 매일경제, 한국경제 등 공신력 있고 검증된 메이저 언론사의 팩트 기반 보도"**만을 필터링하고 요약 인용하도록 지침을 강화하여 정보 신뢰성을 보증했습니다.
- **실시간 12시간 이내 시간 제한 필터 및 한국 시간대(KST) 연동 (최신 업데이트)**:
  - "현재 이 시간부터 최대 12시간 이전의 최신 뉴스만 들려달라"는 요구사항에 맞춰, 시간 엄격 필터 로직을 설계 및 구현했습니다.
  - **KST 기준 실시간 요약**: Gemini API에 뉴스를 요청할 때, 대한민국 표준시(KST - Asia/Seoul) 로컬 시각을 기준시로 명시하고 이 시각으로부터 12시간 이내에 생성된 실시간 정보만 선별해 반환하도록 엄격히 제어했습니다.
  - **모의 뉴스 시뮬레이터**: API Key 없이 로드할 때도, 현재 접속한 한국 시간 기준 12시간 이내의 상대적 시간차(예: 15분 전, 45분 전, 2시간 전 등)를 연산하여 동적으로 타임스탬프를 재생성함으로써 라이브 데이터를 유지합니다.
- **현실성 있는 금융 데이터 반영**:
  - 코스피 지수가 역사상 처음으로 7,000선을 돌파한 상황을 적극 반영하여, 모의 경제 증시 뉴스의 시황 스케일과 이질감 없는 완성도 높은 브리핑 데이터를 구성했습니다.
- **상태 관리 및 환경 보존**:
  - 설정한 관심 분야 체크박스 상태, 프롬프트 문구, API Key, 선택한 목소리 및 낭독 속도는 브라우저의 `LocalStorage`에 반영구적으로 저장되어 앱 재방문 시 자동 복구됩니다.

### [4] 관심 분야 확장 및 설정 창 레이아웃 개편 (최신 업데이트)
- **카테고리 9개 항목으로 대폭 확장**:
  - 기존 4개 카테고리에서 **날씨, 사회, 스포츠, 문화, AI뉴스** 항목을 세분화하여 신설 탑재했습니다.
  - 각 신설 관심 분야별 전용 컬러칩 테마를 지정하여 시각적 직관성을 향상하고, 매칭되는 고품질 모의 뉴스 데이터를 템플릿 세트에 추가했습니다.
- **"전체 선택" 양방향 바인딩**:
  - 설정 모달 내 카테고리 최상단에 **[전체 선택]** 항목을 추가하여 모든 체크박스를 일괄 제어할 수 있도록 했으며, 개별 항목 체크 상태에 따라 전체 선택 체크 여부도 자동으로 갱신되는 양방향 스크립트를 구현했습니다.
- **한글 단독 명칭 제공 및 4열 정렬**:
  - 가독성을 저해하던 괄호 형태의 영문 병기명(예: Politics 등)을 전체 삭제하여 **순수 한글 단독 카테고리명**만 깔끔하게 노출되도록 디자인했습니다.
  - 체크박스 배치를 기존 2열 구조에서 **한 줄에 4개씩 4열 그리드**로 개편해 공간을 절약하고 비주얼을 정돈했습니다. 모바일 등 좁은 화면에서는 2열/1열로 유연하게 반응하도록 미디어 쿼리를 연동했습니다.

### [5] 1열 전체폭 카드 및 카테고리 그룹 헤더 레이아웃 (최신 업데이트)
- **1열 가로 전체폭 뉴스 카드 배치**:
  - 화면의 카드 배치를 2열에서 가로로 길쭉하게 1줄씩만 나오게 구조를 변경하여 한층 가독성 높은 격식 있는 앵커 브리핑 구조로 전면 개편했습니다.
- **카테고리 그룹 헤더 외부 배치 및 중복 표기 제거**:
  - 분야가 같은 카드들이 연속해서 정렬되도록 한 후, **동일 카테고리의 첫 번째 카드 박스 밖의 왼쪽 위**에만 그룹 라벨(예: "증시", "경제")을 한 번만 노출시켰습니다.
  - 이를 통해 카드 내에 카테고리 태그가 기계적으로 중복 노출되는 문제를 완벽히 해결했습니다.
- **제목, 시간, 스피커 아이콘의 1열 수평 정렬**:
  - 카드 시간 표시에 월.일(MM.DD) 정보를 접두사로 합성하여 `"07.13 오전 09:15"` 형태로 시간 정보를 세분화했습니다.
  - 시간 표시 영역을 아래로 내려 **스피커(🔊) 아이콘 바로 왼쪽**에 두고, 뉴스 제목(`card-title`)과 함께 **완벽하게 동일한 수평선상 높이(`align-items: center`)**로 가로 배치했습니다.

### [6] Web Speech API 기반 한국어 TTS 엔진 및 자동 재생 (최신 업데이트)
- 브라우저 내장 음성 합성 시스템(`window.speechSynthesis`)을 제어하여 한국어 음성(`ko-KR`)을 자동 식별하고 드롭다운에 로드합니다.
- **하단 플레이어 퀵 목소리 제어 선택기**: 설정 창을 켜지 않고도 재생 중에 하단 컨트롤 바에서 즉시 목소리를 교체할 수 있도록 목소리 퀵 설정 드롭다운을 탑재했습니다.
- **순차 낭독 및 자동 재생**:
  - 뉴스 요약 수집이 완료되면, 재생 버튼을 번거롭게 누르지 않아도 **자동으로 낭독을 즉시 개시(Autoplay)**하도록 구성했습니다.
  - 전체 재생 시작 시 최초 1회, 오늘 날짜를 읽어주는 인트로 안내 멘트(`"O월 O일 오늘 주요 뉴스를 알려드립니다."`)가 **딱 한 번만 실행**되고 이후에는 반복되지 않도록 제어 장치를 내장했습니다.
- **신경망(Natural/Neural) 보이스 기본값 최우선 지정**:
  - 최고음질 한국어 신경망 보이스인 `Microsoft 선희 Online (Natural) - Korean (Korea)`을 0순위 및 기본 목소리로 최우선 세팅하고, 로컬 스토리지에 타 음성이 저장되어 있었더라도 최초 감지 시 자동 마이그레이션되도록 초기화 코드를 구축했습니다.

### [7] PWA(Progressive Web App) 최적화
- **서비스 워커 (`sw.js`)**: 오프라인 구동이 가능하도록 앱 셸(HTML, CSS, JS, 아이콘)을 캐싱하는 캐시 우선(Cache-First) 전략을 수립했습니다.
- **네트워크 상태 감지**: 네트워크 연결이 유실되면 즉시 상단 메뉴바에 빨간색 '오프라인 모드' 뱃지가 토글되며, 오프라인 시에도 캐싱된 기존 브리핑 뉴스를 안전하게 TTS로 감상할 수 있습니다.
- **설치 배너 & 단독 실행**: `manifest.json` 설정을 완료하고, 브라우저가 설치 가능 상태가 되면 모던 스타일의 '앱 설치 유도 배너'를 화면 상단에 자동으로 띄우고 원클릭 설치 인터랙션을 지원합니다.

### [8] 원클릭 통합 실행기 (start.bat) 개발
- 사용자가 복잡한 터미널 명령어 입력 없이 더블클릭만으로 Vite 로컬 서버 가동과 웹 브라우저 접속을 동시에 자동 수행할 수 있는 윈도우 실행 파일(`start.bat`)을 프로젝트 루트 경로에 제작 및 배포했습니다.

## 2. 사용 도움말 (User Help Guide)

### 🚀 간편 실행 방법 (더블클릭 실행)
1. 프로젝트 루트 폴더(`c:\Temp\Antigrvity\news\`)에 있는 **`start.bat`** 파일을 더블클릭합니다.
2. 자동으로 서버가 구동되고 브라우저가 열려 앱 화면이 바로 로드됩니다.

### 💡 주요 기능 조작 가이드

| 조작 대상 | 단축키 / 방법 | 설명 |
| :--- | :--- | :--- |
| **뉴스 새로고침** | `Ctrl + R` 또는 파일 ➔ 뉴스 새로고침 | 설정된 카테고리와 요구사항에 맞춰 최신 뉴스를 받아옵니다. |
| **설정 창 열기** | `Ctrl + ,` 또는 ⚙️ 퀵 아이콘 / 편집 ➔ 설정(S) | 관심 카테고리 체크, 추가 요구 사항(프롬프트) 입력, API Key 설정 및 목소리/속도 설정을 수행합니다. |
| **테마 변경** | 헤더 우측 달/해 버튼 탭 | 클릭 시 즉각 라이트 모드(기본)와 다크 모드 사이를 교체합니다. |
| **목소리 변경** | 하단 우측 목소리 퀵 드롭다운 | 설정 창을 열 필요 없이 하단 오디오 바에서 다이렉트로 목소리를 변경 연동합니다. |
| **전체 뉴스 듣기** | 하단 플레이어 `재생(▶)` 클릭 | 전체 뉴스 목록을 첫 번째 카드부터 마지막 카드까지 한국어로 순차 낭독합니다. (최초 1회 오프닝 멘트 포함) |
| **특정 뉴스만 듣기**| 개별 카드 우측 상단 `🔊` 클릭 | 선택한 특정 뉴스만 독립적으로 즉시 낭독합니다. (인트로 생략) |
| **PDF로 저장하기** | `Ctrl + P` 또는 파일 ➔ PDF 저장 | 낭독 중인 카드 및 플레이어 컨트롤을 제외한 깔끔한 문서 레이아웃으로 뉴스 브리핑을 PDF 인쇄 저장합니다. |
| **도움말 열기** | `F1` 또는 도움말 ➔ 도움말 보기 | 현재 도움말 팝업을 열어 단축키 및 기능 안내를 확인합니다. |

### 📱 모바일 및 데스크톱 앱 설치 요령
1. **데스크톱 (크롬/엣지)**:
   - 앱에 접속하면 화면 상단에 **'설치하기' 배너**가 나타납니다. 이를 누르면 시스템에 설치되어 독립된 창 모드로 사용할 수 있습니다.
2. **모바일 (Android)**:
   - 크롬 브라우저 우측 상단 메뉴 ➔ **'홈 화면에 추가'**를 선택하면 바탕화면에 설치됩니다.
3. **모바일 (iOS/Safari)**:
   - 하단 공유 아이콘 ➔ **'홈 화면에 추가'**를 터치하면 앱 서랍에 저장됩니다.
