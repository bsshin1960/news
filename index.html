뉴스 및 TTS 요구사항 설정

선택된 언론사의 기사로 제한하여 뉴스를 검색합니다. 모두 해제하거나 모두 선택하면 전체 언론사에서 검색합니다.
API가 실패하거나 비어 있으면 최신 뉴스 RSS로 자동 보완합니다.
OpenAI API를 사용할 때만 필요합니다. 로컬 브라우저 저장소에 저장됩니다.
Groq 무료 API를 사용해 RSS 기사 원문을 빠르게 4줄로 요약합니다.
Google AI Studio에서 발급받은 API 키를 입력하면 실시간 구글 검색 뉴스 브리핑이 동작합니다.
각 뉴스 항목을 몇 글자 정도로 작성하고 들려줄지 설정합니다. 권장 범위는 150~2000자입니다.
뉴스 낭독의 구성 단계를 선택합니다. 헤드라인은 제목을 의미합니다.

사용 방법 및 안내

💡 주요 조작 방법

  • 오늘의 뉴스 브리핑 받아오기: 상단 메뉴의 파일 -> 뉴스 새로고침 (Ctrl+R)을 선택하면, 설정한 필터에 맞춰 최신 뉴스가 요약 제공됩니다.
  • 관심사 및 뉴스 스타일 설정: 상단 메뉴의 편집 -> 설정 (Ctrl+,)을 눌러 관심사 체크박스를 선택하고, 원하는 요약 스타일(프롬프트)을 입력할 수 있습니다.
  • 뉴스 TTS 낭독:
    • 하단 재생 제어 패널의 재생 (▶) 버튼을 누르면 전체 뉴스를 처음부터 순차적으로 읽어줍니다.
    • 각 뉴스 카드의 오른쪽 하단에 있는 듣기 버튼을 누르면 해당 뉴스를 개별적으로 들을 수 있습니다.
    • 현재 낭독 중인 카드는 테두리가 화려한 네온 컬러로 강조되며, 화면이 자동으로 스크롤됩니다.
  • PDF로 저장하기: 상단 메뉴의 파일 -> 브리핑 PDF 저장 (Ctrl+P)을 누르면 깔끔하게 다듬어진 인쇄 스타일로 뉴스 브리핑을 보관할 수 있습니다.

📱 PWA 설치 방법

  • 데스크톱: 주소창 우측의 설치 아이콘을 클릭하거나 상단의 '설치하기' 배너를 누르면 독립형 앱으로 설치됩니다.
  • 모바일 (Android/iOS): 크롬의 '홈 화면에 추가' 메뉴나 사파리의 '홈 화면에 추가'를 터치하여 앱처럼 바로 실행할 수 있습니다.
